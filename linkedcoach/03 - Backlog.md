# Backlog — LinkedCoach

> Features planejadas para após o MVP. Ordenadas por prioridade percebida.

---

## Pós-MVP — Alta Prioridade

- [ ] **Compartilhamento de post** — gerar link público para um post gerado (sem auth para visualizar)
- [ ] **Regenerar post** — botão para gerar uma nova versão sem apagar a atual
- [ ] **Tons predefinidos** — dropdown com tons populares ("CEO arrependido", "Empreendedor de primeira viagem", "Motivacional genérico")
- [ ] **Contador de gerações** — exibir quantos posts o usuário já gerou no total

## Pós-MVP — Média Prioridade

- [ ] **Deletar post do histórico** — botão de lixeira em cada item do histórico
- [ ] **Favoritar post** — marcar posts como favoritos para não serem deletados no limite de 20
- [ ] **Modo escuro / claro** — toggle de tema (hoje é sempre escuro)
- [ ] **Analytics de uso** — painel interno: gerações por dia, usuários ativos

## Ideias Brutas (não priorizadas)

- Templates de posts por tema (liderança, demissão épica, primeiro emprego, falência gloriosa)
- Geração em lote (5 variações do mesmo post de uma vez)
- Integração direta com LinkedIn API para postar (requer OAuth LinkedIn)
- Modo "batalha" — dois posts gerados com tons diferentes, usuário vota no melhor
- Exportar histórico como PDF ou CSV
